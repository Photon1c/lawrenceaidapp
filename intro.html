Lawrence: A Legal Administrative Specialist Cobot

This app is designed to streamline the average daily workflow of an LAS.

It performs the following two main functions, each accessible in its own Microsoft Teams Tab:

1.- Document Manager
The document manager tab receives incoming emails, follows URLs to download attachments, downloads them,
renaming them based on information obtained in the email subject, body, or additionally added OCR programs.
It then forwards the email to a selected party notifying them of the task completion.

2.- Lawrence Chatbot
Lawrence is built from a Microsoft QnA database that contains predicted questions along with the suggested
bot responses to these. The entire app is in constant development and recommended add ons that are in sandbox
development include: OpenAI support, Tensorflow machine learning module support, total Outlook integration.
